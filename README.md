## 💡 프로젝트 개요
해당 프로젝트는 분당우리교회 웹사이트를 반응형으로 리뉴얼하였습니다.
개인프로젝트로서 디자인, 퍼블리싱을 모두 100% 개인작업으로 진행하였습니다.

## ✨ 개선점

- 드롭메뉴 스타일로 하위항목 정리, 6개로 메뉴구성 축소
- 아이콘을 추가하여 글자수를 줄임
- 대체 텍스트 사용하여 시각적 정보 취약 사용자를 위한 기능 개선
- 중요한 항목을 퀵메뉴로 변경하여 어디서든 접근성을 높임
- 가운데 정렬로 균형을 맞춤
