## 프로젝트 개요
해당 프로젝트는 분당우리교회 웹사이트를 반응형으로 리뉴얼하였습니다.
개인프로젝트로서 디자인, 퍼블리싱을 모두 100% 개인작업으로 진행하였습니다.




## 사용툴
- VS Code
- Figma
- Illustrator
- Photoshop

## UI/UX 디자인

- 레이아웃 선정 [📐](https://www.nngroup.com/articles/f-shaped-pattern-reading-web-content-discovered/) - F자 패턴
- 터치스크린 아이콘 적정크기 [👍](https://uxplanet.org/checklist-for-using-icons-in-ui-acb6dee36e8a) - 7-10mm
- 검색 버튼 [🔍](https://designmodo.com/ux-search-website/) - 중요성 및 적정위치
- 모바일 헤더 [📌](https://cafemedia.com/best-practices-for-sticky-elements-in-ux-design/) - 고정요소 관련 UX 심리학
- 슬라이더 디자인[◀▶](https://www.nngroup.com/articles/gui-slider-controls/) -슬라이더 사용 조건, 탭버튼필요성

## 목표

- 전연령층 및 장애인의 사용성 고려
- 어떤 기기든 최적의 웹 페이지를 볼 수 있도록 함
- 방문자를 위한 디자인





사용자는 먼저 콘텐츠 영역의 위쪽 부분을 수평 이동으로 읽습니다(Nielsen, 2006). 그러므로, 세로형태의 메뉴바 보다는 가로형의 메뉴바를 사용하여 보다 빠르게 원하는 컨텐츠를 찾도록 하고자 합니다.

- 가로형 메뉴바 사용
- 좁은 레이아웃 구조로 정보 검색와 사용에 어려움 

- 적절한 아이콘사용으로 데이터 인식 속도 향상
